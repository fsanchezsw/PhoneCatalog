# PhoneCatalog
Phone catalog app with basic Node APIREST and Angular in Front-end side

# Installation

For NodeJS API:

```sh
npm install:api

```

For Angular APP:

```sh
npm install:app
```

# Development setup

Run the project in two different terminals, one for API and one for APP. First terminal:

```sh
npm run api
```

Second terminal:

```sh
npm run app
```
